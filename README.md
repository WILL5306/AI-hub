
# [AI-hub]

## 📖 项目简介
这是一个基于 **Electron** 开发的桌面应用程序。
AI-hub 是一款基于 Electron 构建的现代化桌面应用程序。它旨在为用户提供多种AI及实用工具体验。以轻量级的体积和流畅的交互，帮助你更轻松地完成工作，告别繁琐的操作流程

## 🛠️ 技术栈
- **核心框架**: Electron
- **运行时**: Node.js
- **界面**: HTML5, CSS3, JavaScript (或 Vue/React)

## 📂 目录结构
```text
├── main.js        # 主进程：窗口管理与系统交互
├── preload.js     # 预加载：安全桥接
├── index.html     # 界面结构
├── renderer.js    # 界面逻辑
├── styles.css     # 界面样式
└── package.json   # 依赖与脚本配置
```
## 🚀 使用
1.安装 Node.js
2.安装依赖，在项目根目录下运行终端命令：
```
npm install
```
3.运行
```
npm start
```
即可出现界面

4.打包
```
npm run dist
```
打包后安装包
